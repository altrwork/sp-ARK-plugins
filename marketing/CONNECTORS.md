# Connectors

Plugin skills use `~~category` as a placeholder for whatever tool the user connects in that category.

Both connectors below are Google-hosted MCP servers. Authentication is handled through the claude.ai integrations page — no local server or credentials setup required.

| Category | Tool included | MCP URL | Alternatives |
|---|---|---|---|
| `~~email` | Gmail | `https://gmailmcp.googleapis.com/mcp/v1` | Outlook, Superhuman |
| `~~spreadsheet` | Google Drive / Sheets | `https://drivemcp.googleapis.com/mcp/v1` | Airtable, Notion database |

## Setup

1. Go to **claude.ai → Settings → Integrations**
2. Connect your **Gmail** account
3. Connect your **Google Drive** account
4. Sign in as the Google account that owns the CEO's inbox and the distribution list Sheet
